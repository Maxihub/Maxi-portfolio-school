    install node.js.
    run "npm install -g gatsby-cli" in the terminal
    Open the project and run "npm install" in the frontend folder
    in portfolio-backend\extensions\upload\config\settings.json
    change these variables to your own or delete
    now run "npm install" in the backend folder
    stay in backend and run "npm run develop"
    click the link "http://localhost:1337/admin" or type in a web browser
    create an account and insert information about yourself.
    save all changes
    now run gatsby develop in the frontend folder.
    click the link "http://localhost:8000/" or type in a web browser
    now you can see what you're not satisfied with and which variables you need
    to customize to make the portfolio your own
